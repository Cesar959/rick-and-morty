Seja bem-vindo! Ao projeto Rick and Morty, por meio do painel do projeto você pode conhecer os personagens acamados da famosa série. Para o desenvolvimento desse projeto foi utilizado a The Rick and Morty API, uma API pública que retorna os dados dos personagens da série.
A criação do Layout foi feita baseando-se na série, a ideia serie fazer referência a ela nas cores e alguns elementos posicionados na tela. Para o desenvolvimento da interface foi utilizado o Figma para isso.
